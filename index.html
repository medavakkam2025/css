Understanding Flexbox in CSS

Key Concepts and Interview Questions

Presented by You

What is Flexbox?

  • Flexbox (Flexible Box Layout) is a CSS layout module.
  • Arranges items in a one-dimensional space (horizontally or vertically).
  • Helps in aligning, distributing, and spacing elements dynamically.

Main Components of Flexbox

  • Flex Container: Parent element with display: flex or display: inline-flex.
  • Flex Items: Direct children of a flex container.

Making an Element a Flex Container

display: flex;

justify-content vs. align-items

  • justify-content: Aligns items along the main axis (horizontal by default).
  • align-items: Aligns items along the cross axis (vertical by default).

flex-direction

Defines the direction of the main axis.

  • row (default): left to right
  • row-reverse: right to left
  • column: top to bottom
  • column-reverse: bottom to top

flex-wrap

Controls whether flex items wrap onto multiple lines.

  • nowrap (default): All items in one line.
  • wrap: Items wrap to the next line if needed.
  • wrap-reverse: Items wrap in reverse direction.

order Property

Defines order of flex items in the container regardless of HTML source.

Items with lower values appear first.

flex Shorthand Property

A shorthand for flex-grow flex-shrink flex-basis.

flex: 1 0 200px;

This means: grow if needed (1), don't shrink (0), base size 200px.

align-self

Overrides align-items setting for individual flex items.

Accepts values like auto, flex-start, center, flex-end, stretch.

gap in Flexbox

Defines spacing between flex items.

gap: 20px;

Adds 20px space between each item horizontally or vertically based on direction.

Conclusion

  • Flexbox is a powerful layout tool in CSS.
  • Understanding its properties and values is essential for modern web design.
